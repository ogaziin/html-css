Apresentando GitSync Enterprise

Mais moderno, mais simples, mais rápido.

Chega de complexidades. GitSync Enterprise chegou para atender aos usuários que desejam um layout e um funcionamento bem simples, que nem ao menos requer um conhecimento técnico especial.

Requisitos

  • Processador: Intel Core i5 ou equivalente
  • Memória: 4 GB de RAM
  • Armazenamento: 10 GB de espaço livre no disco
  • Sistema Operacional: Windows 10, macOS 11 ou Linux

Outras versões: GitSync Advanced

Uma versão mais complexa. Criada para aqueles que gostam do padrão ou da complexidade. É a edição clássica. Mais leve, porém requer conhecimentos técnicos mais avançados.

Requisitos

  • Processador: Intel Core i3 ou equivalente
  • Memória: 1 GB de RAM
  • Armazenamento: 3 GB de espaço livre no disco
  • Sistema Operacional: Windows 7, macOS 8 ou Linux
